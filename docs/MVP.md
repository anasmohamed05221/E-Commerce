***MVP Scope***
✅ MVP (what we will be built now)

- Customer can browse products (list + details)
- Customer can filter products (by price, category)
- Customer can manage cart (add/remove/update qty)
- Customer can checkout → order created from cart, payment mocked (not a real payment service)
- Customer can view my orders (list + details including status)
- Admin can create/update/delete products (basic)
- Inventory decreases when order is created (or reserved — choose one)

--------------------------------------

❌ Not MVP (later)

- coupons/discounts
- reviews/ratings
- wishlist
- returns/refunds
- shipping/delivery tracking
- payment gateway integration (Stripe, etc.)
- multi-vendor/sellers
